# 台灣武漢肺炎確診地圖


###### 網站連結
* https://wuhan-virus-taiwan.firebaseapp.com/

###### 操作說明
* 查看確診分布
 * 1.點選區塊，出現範圍座標
 * 2.點選座標
 * 3.出現確診病例詳細資料

* 新增確診案例
 * 1.點選確診日期，出現日曆表
 * 2.點日曆表確診時間(預設為當天日期)
 * 3.選擇確診地區，地圖經緯度自動調整為當區範圍
 * 4.點選地圖微調經緯度位置
 * 5.輸入確診案例內容
 * 6.備註部分輸入目前確診案例，若有特殊情形EX:已痊癒、以確診、社區感染等
 * 7.提交案例資料
 * 8.完成!
